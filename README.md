<H1>오늘의 질문</H1>

매일 변하는 다양한 질문에 따라 다양한 답변을 함으로써 흥미를 이끌어내 적극적인 어플 사용 추구하고
다른 이용자의 다양한 의견을 보면서 아이디어 또는 영감을 받을 수 있고 팔로우 기능을 통한 유대감을 형성합니다.

프로그래밍 언어 : Kotlin, Python <br>
DB : SQLite <br>
서버 : flask <br>
IDE : 안드로이드 스튜디오 <br>
빌드도구 : Gradle <br>
라이브러리 : Gson, Retrofit(REST API), Coil, Paging3 <br>
참고 : chat GPT <br>
